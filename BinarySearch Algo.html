Binary Search Patterns Cheat Sheet

0. Master Rule: When to Use ans / res

Need BEST answer? → Use ans/res Need ANY answer? → Return immediately

Use ans (or res) whenever you need to keep searching after finding a valid answer.

Examples:

  • First Occurrence
  • Last Occurrence
  • Lower Bound
  • Upper Bound
  • Search Insert Position
  • First Bad Version
  • Minimum Days to Make Bouquets
  • mySqrt
  • Aggressive Cows
  • Maximum Minimum Distance
int ans = -1;

while (left <= right) {

    int mid = left + (right - left) / 2;

    if (condition(mid)) {

        ans = mid; // Save current answer

        // First True / First Occurrence
        // right = mid - 1;

        // Last True / Last Occurrence
        // left = mid + 1;
    }
    else {
        // Move opposite direction
    }
}

return ans;

First True Pattern

F F F F T T T T ^ First True 
if (condition(mid)) {
    ans = mid;
    right = mid - 1;
} else {
    left = mid + 1;
}

Last True Pattern

T T T T F F F F ^ Last True 
if (condition(mid)) {
    ans = mid;
    left = mid + 1;
} else {
    right = mid - 1;
}

When NOT to Use ans/res

if (arr[mid] == target)
    return mid;

Exact Binary Search does not need ans/res because the search ends immediately when the target is found.

1. Classic Binary Search

while (left <= right) {
    int mid = left + (right - left) / 2;

    if (arr[mid] == target)
        return mid;
    else if (arr[mid] < target)
        left = mid + 1;
    else
        right = mid - 1;
}
return -1;

8. Search in Rotated Sorted Array

while (left <= right) {
    int mid = left + (right - left) / 2;

    if (arr[mid] == target)
        return mid;

    if (arr[left] <= arr[mid]) {

        if (target >= arr[left] && target < arr[mid])
            right = mid - 1;
        else
            left = mid + 1;

    } else {

        if (target > arr[mid] && target <= arr[right])
            left = mid + 1;
        else
            right = mid - 1;
    }
}

9. Find Minimum in Rotated Sorted Array

while (left < right) {
    int mid = left + (right - left) / 2;

    if (arr[mid] > arr[right])
        left = mid + 1;
    else
        right = mid;
}

return arr[left];
